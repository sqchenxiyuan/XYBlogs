# 简单工厂模式(Simple Factory Pattern)

本文简单讲述简单工厂模式

## 功能

**简单工厂模式**其实算不上设计模式，但是由于十分简单好用，所以经常被使用到，它是一种创建类的设计模式，对对象的创建进行的了封装

简单工厂模式也十分容易理解，就是传入一个容易理解的参数，来获取对应的复杂对象，就像工厂一样我想要什么，工厂就会给我生成什么出来
